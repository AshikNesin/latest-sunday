# latest-sunday

> Get the latest sunday of the week

## Install

```
$ npm install --save latest-sunday
```


## Usage

```js
const latestSunday = require('latest-sunday');

const latestSundayDt = latestSunday();

//=> 'Sun Oct 21 2018' (It'll return the recent sunday)
```
